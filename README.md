# CivicPulse - Crowd-Verified City Problem Radar

A geospatial full-stack platform that clusters public civic reports by location and confidence.

## Stack
- Node.js + Express + SQLite backend
- Surreal map-first frontend (HTML/CSS/JS)

## Run Backend
```bash
cd backend
npm install
npm start
```

## Run Frontend
Open `frontend/index.html` in browser.

## Resume Value
- Geospatial clustering logic
- Actionable confidence scoring
- Highly differentiated map-driven product design
